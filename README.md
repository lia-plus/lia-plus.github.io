# LIA+ Rede de Laboratórios

Site institucional da rede LIA+ (Laboratório de Inteligência Artificial e Tecnologias Disruptivas), vinculada à ESALQ/USP.

🌐 **Site:** [lia-plus.github.io](https://lia-plus.github.io)

## Sobre este repositório

Este repositório contém o código-fonte do site guarda-chuva da rede LIA+, construído com Jekyll e tema Cayman via GitHub Pages.

## Estrutura

```
lia-plus.github.io/
├── index.md          # Página principal
├── _config.yml       # Configurações do Jekyll
└── assets/           # Imagens e arquivos estáticos
```

## Contribuindo

Commits seguem a convenção:

- `feat` — nova página ou seção
- `content` — alteração de texto existente
- `assets` — imagens e arquivos estáticos
- `style` — CSS e layout
- `fix` — correções de links, imagens ou CSS
- `config` — _config.yml e configurações do Jekyll
- `docs` — este README
- `chore` — manutenção geral
